다음 문장은 유해할 수 있는 표현을 포함하고 있을 수 있습니다. 아래 지침에 따라 해당 문장을 적절하게 정제해주세요.

요구사항:
- 원문의 의도와 감정은 최대한 보존하면서, 유해하거나 부적절한 표현만 정제해주세요.
- **문장이 너무 유해해서 의미 보존이 어려운 경우, 원문과 유사한 의도를 담은 대체 가능한 문장으로 바꿔주세요.
- COT (Chain of Thought) 방식으로 판단하고 수정해주세요:
  - 어떤 단어/표현이 문제가 되는지 설명
  - 왜 그 표현이 문제인지 (예: 혐오, 비속어, 사회적 편견 등)
  - 어떤 방식으로 대체하는 것이 적절한지 (예: 의미를 유지하되 표현을 완화)
- 대상 문장은 어린이 및 청소년도 안전하게 읽을 수 있어야 합니다.
- 정제된 문장은 **자연스럽고 문맥상 어색하지 않아야** 합니다.

출력 형식은 반드시 ""아래의 JSON 형식만 사용해주세요"".  
추가 설명, 해설, 메타 정보 등은 **절대 포함하지 마세요"".

Output 형식:
json
{
  "문장중 유해한 단어들": ["..."],
  "대체 제안 형식": "...",
  "대체 문장": "..."
}

json 예시
{
  "문장중 유해한 단어들": ["개쌔끼"],

  "대체 문장": "왜 그렇게 말하는 거야, 기분이 안 좋았으면 말로 해."
